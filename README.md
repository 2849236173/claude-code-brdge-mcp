# claude-code-brdge-mcp
用来跳出ai所修改代码部分差异窗口的插件

安装：
1. 选中idea, 从磁盘加载插件, 选中这个压缩包
2. 给对应ai程序添加这个mcp, 如claude-code:
claude mcp add --transport sse claude-bridge http://127.0.0.1:3000/sse
